┌─────────────────────────────────────────────────────────────────────────────┐
│                              DEFINITIONS                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   1. DNA                                                                    │
│      All living beings have DNA.                                            │
│      Four letters: A, T, C, G.                                              │
│      Human: 3 billion letters. Chimp: 3 billion. Banana: 500 million.       │
│                                                                             │
│   2. GENETIC DISTANCE (X)                                                   │
│      Count the differences between two DNA samples.                         │
│                                                                             │
│         A T C G A T C G                                                     │
│         A T C G A T T G                                                     │
│                     ↑                                                       │
│         X = 1                                                               │
│                                                                             │
│   3. GENERATIONAL DISTANCE (N)                                              │
│      Generations back to common ancestor.                                   │
│                                                                             │
│   4. MUTATION                                                               │
│      One letter changes. About 100 per generation.                          │
│                                                                             │
│   5. INHERITANCE                                                            │
│      Half from mother. Half from father. Plus new mutations.                │
│      Inheritance reshuffles. Mutations create.                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                              THE INSIGHT                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   If there are differences, there is a common ancestor.                     │
│   The more differences, the more generations back.                          │
│   No limit to how different. No limit to what form.                         │
│   Completely independent creation is possible. This is the key point.       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                              CONCLUSION                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Everything is one. And not one.                                           │
│   One because of a common origin, a shared genetic system, a shared history.│
│   Not one because of the ability to express completely opposite forms.      │
│                                                                             │
