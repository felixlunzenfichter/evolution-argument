│                              DEFINITIONS                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   1. DNA                                                                    │
│      All living beings have DNA.                                            │
│      DNA can be represented as a chain of four letters:                     │
│      A, T, C, G.                                                            │
│                                                                             │
│   2. GENETIC DISTANCE (X)                                                   │
│      We can count the differences between two DNA samples.                  │
│                                                                             │
│         Person A:   A T C G A T C G                                         │
│         Person B:   A T C G A T T G                                         │
│                                 ↑                                           │
│                            1 difference                                     │
│                                                                             │
│         X = differences / total length = 1/8 = 12.5%                        │
│                                                                             │
│   3. GENERATIONAL DISTANCE (N)                                              │
│      The smallest number of generations back                                │
│      until two beings share a common ancestor.                              │
│                                                                             │
│   4. MUTATION RATE IS BOUNDED                                               │
│      There is a maximum number of mutations per generation.                 │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
